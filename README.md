Files Overview:

- `arcade-chart-sample.js`: Line chart showing streamgage height over the previous 30 days in a Map Viewer popup using Arcade
- `slides-samples.js`: Various Arcade snippets found in presentation slides (conditional formatting, returning HTML components, etc.)
- `monopoly_card.js`: Example using CSS, HTML, and Arcade in a Dashboard list
- `css_carousel.html`: Example using CSS and HTML to create a scrolling carousel without Javascript
- `coxcomb_chart.js`: Example using HTML, CSS, Arcade, and SVG to create a coxcomb chart within a Dashboard list
- `gantt_chart.js`: Example using HTML, CSS, Arcade, and SVG to create a Gantt chart within a Dashboard table
- `html_insert.js`: Example using HTML in Arcade to show how HTML is sanitized differently within Map Viewer popup
- `Unlocking the Power of Configurable Apps with HTML, CSS, and ArcGIS Arcade.pptx`: Session slides
